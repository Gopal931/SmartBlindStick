🦯 Smart Blind Stick

A Smart Blind Stick is an assistive device designed to help visually impaired people navigate safely and independently. This project integrates multiple sensors and communication modules to detect obstacles, prevent accidents, and provide emergency assistance.

🚀 Features
🔍 Obstacle Detection
Detects objects in front using an ultrasonic sensor and alerts the user.
💧 Water Detection
Identifies wet or slippery surfaces to prevent falling.
🔊 Real-Time Alerts
Instant feedback using buzzer or vibration.
🚨 Emergency SOS Button
A dedicated button that, when pressed:
Sends an SMS alert to a caretaker
Makes an emergency call automatically
📡 GSM Communication (SIM900A Module)
Enables wireless communication without internet using a SIM card.
⚡ Portable & Lightweight
Designed for daily usability.



🛠️ Technologies Used

Microcontroller: Arduino / NodeMCU
Sensors:
Ultrasonic Sensor
Water Detection Sensor
Communication Module:
SIM900A GSM Module
Other Components:
Push Button (SOS)
Buzzer / Vibration Motor
Battery
Connecting wires



⚙️ How It Works
Obstacle Detection
Ultrasonic sensor measures distance continuously
Alerts user if an object is too close
Water Detection
Sensor detects water on the ground
Alerts user to avoid slipping
Emergency SOS System
User presses the SOS button
Microcontroller sends command to SIM900A module
Module:
📩 Sends SMS to caretaker
📞 Initiates a call automatically
